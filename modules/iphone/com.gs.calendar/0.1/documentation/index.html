Calendar Titanium Module for iOS (v0.1)

Description

This module aims to provide access to EventKit and EventKitUI functionality on iOS.

Currently it allows you to add an event to the user's default calendar with the iOS built-in event edit dialog, but more features are being added soon (the ability to search for existing events, ability to edit or delete existing events, monthly calendar view, are a few of the planned features).

Compatibility

The module was tested with Titanium SDK 1.7.2 on iOS SDK 4.3.

Even though EventKit framework is only available for iOS 4 and newer, your application should still work with older iOS versions despite using this module - check the included example file for more information.

Installing the module

You can either install the module locally under a particular project, or inside the Titanium Mobile SDK folder (that way it will be accessible by all Titanium Mobile projects).

Local installation for a single project

Put the zip file inside your project's root folder (same folder where the tiapp.xml file for your app is located).

Unzip the file - this should put the module files inside the modules/iphone/com.gs.calendar/0.1 folder.

Installation under Titanium Mobile SDK (if used for more projects)

Put the zip file inside the /Library/Application Support/Titanium folder (the one on the root of your Mac HD, not under your home folder).

You do not need to unzip the file (it will get unzipped automatically when you build a project), but if you do unzip it, the module files will be inside the modules/iphone/com.gs.calendar/0.1 folder.

Registering the module for your application

Register the module with your application by editing tiapp.xml and adding the module under the modules tag.

Example:

<modules>
    <module version="0.1">com.gs.calendar</module>
</modules>

Accessing the module in your code

To access this module from JavaScript, you would do the following:

var calendar = require("com.gs.calendar");

The calendar variable is a reference to the Module object.

Reference
